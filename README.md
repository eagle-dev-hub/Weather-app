Weather Dashboard

This is a simple weather application I built to practice working with APIs, asynchronous JavaScript, and responsive web design. It lets users select a city and instantly view its current weather information in a clean, modern interface.

Building this project helped me strengthen my understanding of the Fetch API, DOM manipulation, and creating responsive layouts with CSS.

Features

- View current weather for different cities
- Displays:
  - Weather condition
  - Temperature
  - Feels-like temperature
  - Humidity
  - Wind speed
  - Wind gust
- Responsive design for mobile and desktop
- Clean and modern user interface
- Fetches live weather data using an API

Built With

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await

What I Learned

While building this project, I practiced:

- Working with REST APIs
- Using "async" and "await"
- Handling asynchronous data
- Updating the DOM dynamically
- Organizing JavaScript into reusable functions
- Building responsive layouts with Flexbox and CSS Grid
